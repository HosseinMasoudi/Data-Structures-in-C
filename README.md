# Data-Structures-in-C

This repository is a study and practice space for the book *Fundamentals of Data Structures in C* by Ellis Horowitz and co-authors.

The goal of this repo is to collect the code, exercises, and answers from the book in one place, written in C and organized for easy review and practice. It is meant to help with learning the core ideas behind data structures by implementing them directly and working through the book’s problems step by step.

## What you will find here

- Implementations of data structures in C
- Solutions to selected book exercises
- Practice code for understanding the ideas more deeply

## Purpose

This project is mainly for:

- Learning data structures through hands-on coding
- Keeping book notes and solutions in one repository
- Building a clear reference for future study and revision

## Notes

The code in this repository is written as part of my personal practice with the book and may continue to grow as I work through more chapters and problems.
